Comment obtenir le wifi de l'IUT ?

Pour avoir accès aux wifi du département il suffit d'aller sur le site de la BU de Franche-Comté. Vous apprendrez ici comment vous connectez facilement.

Avant de commencer ce petit tuto. Assurez-vous de remplir ces prérequis :

  • une carte réseau sans fil correctement installée avec les derniers pilotes compatibles Windows
  • Le nom du réseau (SSID) ainsi que la clé WEP qui vous auront été fournis au préalable
  • le client DHCP de votre ordinateur vous permettant d'obtenir automatiquement une adresse IP
  • un navigateur Web supportant l'encryptage SSL des données
  • Etre sous windows

La première étape est de télécharger l'utilitaire SecureW2

L'étape suivante est simplement de suivre les différentes étapes du téléchargement que vous propose cette utilitaire
